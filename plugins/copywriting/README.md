# COPYWRITING

## Tags
DIRECT RESPONSE, SALES COPY, HEADLINES, TODD BROWN, GARY HALBERT, JANUARY_IMPORT

## Description
This skill provides a comprehensive framework for writing high-converting direct response copy based on methodologies from Todd Brown, Gary Halbert, Eugene Schwartz, Frank Kern, and Dan Kennedy.

**How This Skill Helps:**
- Applies Todd Brown Big Idea and Unique Mechanism frameworks
- Uses Gary Halbert 6 Types of Leads for powerful openings
- Implements Eugene Schwartz awareness levels for message matching
- Stacks proof and creates irresistible offers that overcome objections

**When to Use:** Use when writing sales pages, VSLs, email campaigns, ads, or any copy where driving conversions is critical.
