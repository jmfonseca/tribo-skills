# AB SPLIT TEST ENGINEERING

## Tags
A/B TESTING, CONVERSION OPTIMIZATION, STATISTICAL ANALYSIS, CRO, DATA-DRIVEN MARKETING, JANUARY_IMPORT

## Description
This skill provides a structured approach to A/B split test engineering, enabling you to systematically improve your marketing campaigns and website performance.

**How This Skill Helps:**
- Prioritizes tests using ICE scoring to focus on highest-impact opportunities
- Calculates proper sample sizes and statistical significance thresholds
- Documents test hypotheses and isolates variables for clean results
- Analyzes conversion lift to quantify the business impact of winning variations

**When to Use:** Use when you need to optimize landing pages, email subject lines, ad creatives, or any marketing element where data-driven decisions can increase conversions.
