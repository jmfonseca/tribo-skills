# ADVERTORIAL WRITING

## Tags
NATIVE ADVERTISING, ADVERTORIALS, CONTENT MARKETING, TABOOLA, OUTBRAIN, JANUARY_IMPORT

## Description
This skill produces compelling advertorial content that reads like a native article while effectively persuading the reader to take action.

**How This Skill Helps:**
- Matches the editorial voice and style of the target publication platform
- Uses story-based pre-framing to engage readers emotionally before the pitch
- Ensures platform compliance with Taboola, Outbrain, and Facebook guidelines
- Integrates soft CTAs that feel natural within the narrative flow

**When to Use:** Use when running native advertising campaigns on content discovery platforms or creating sponsored content for publications.
