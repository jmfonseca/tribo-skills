This skill enables Claude to act as a Search Engine Marketing (SEM) specialist, creating, managing, and optimizing paid search campaigns.

**How This Skill Helps:**
- Develops keyword strategies aligned with search intent
- Writes compelling ad copy that drives clicks and conversions
- Optimizes landing pages for Quality Score
- Manages bids and budgets for profitability

**When to Use:** Use when running Google Ads campaigns, managing paid search budgets, or optimizing PPC performance.
