# UPSELL DOWNSELL SCRIPTING

## Tags
UPSELLS, DOWNSELLS, ORDER BUMPS, RUSSELL BRUNSON, OTO STRATEGY, JANUARY_IMPORT

## Description
This skill enables the creation of high-converting upsell and downsell scripts based on Russell Brunson One-Time Offer (OTO) methodology.

**How This Skill Helps:**
- Structures one-time offers that increase average order value
- Creates order bump offers that complement main purchases
- Develops downsell strategies for non-buyers
- Writes persuasive scripts without excessive pressure

**When to Use:** Use when optimizing checkout flows, adding upsell sequences, or increasing average order value.
