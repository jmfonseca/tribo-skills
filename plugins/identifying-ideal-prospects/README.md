# IDENTIFYING IDEAL PROSPECTS

## Tags
IDEAL CUSTOMER PROFILE, TARGET AUDIENCE, LEAD QUALIFICATION, PROSPECTING, B2B TARGETING, JANUARY_IMPORT

## Description
This skill pinpoints ideal clients and builds a strategic plan to engage them effectively through data-driven prospecting.

**How This Skill Helps:**
- Develops detailed ideal customer profiles beyond demographics
- Identifies where your prospects spend time online
- Creates qualification frameworks to focus on best-fit leads
- Builds systematic prospecting processes

**When to Use:** Use when defining target audiences, building prospecting lists, or improving lead quality.
