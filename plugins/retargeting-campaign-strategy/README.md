# RETARGETING CAMPAIGN STRATEGY

## Tags
RETARGETING, REMARKETING, AUDIENCE SEGMENTATION, SEQUENTIAL ADS, FREQUENCY CAPPING, JANUARY_IMPORT

## Description
This skill creates a comprehensive retargeting campaign strategy based on teachings of Jon Penberthy and Frank Kern.

**How This Skill Helps:**
- Segments audiences based on behavior and intent
- Develops sequential ad creative that tells a story
- Implements frequency capping to prevent ad fatigue
- Creates exclusion audiences to optimize spend

**When to Use:** Use when building retargeting campaigns, recovering abandoned carts, or re-engaging website visitors.
