# Tribo Skills

84 marketing skills for Claude - copywriting, funnels, email, SEO, and more. All 84 skills ship as a single plugin, `tribo-marketing`.

## Installation

### Claude Code / Cowork

```bash
# Add the marketplace
claude plugins marketplace add https://github.com/jmfonseca/tribo-skills

# Install the plugin (includes all 84 skills)
claude plugins install tribo-marketing@tribo-skills
```

In Cowork, use **Add marketplace** and enter `jmfonseca/tribo-skills`, then install `tribo-marketing`.

### Install from local clone

```bash
# Clone the repository
git clone https://github.com/jmfonseca/tribo-skills.git

# Add the local marketplace
claude plugins marketplace add ./tribo-skills

# Install the plugin
claude plugins install tribo-marketing@tribo-skills
```

## Usage

Once installed, the skills activate automatically when your request matches a skill's
description. You can also browse what's available — every skill lives under
`tribo-marketing/skills/<name>/`.
